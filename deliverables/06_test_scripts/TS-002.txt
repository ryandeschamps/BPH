TEST SCRIPT: TS-002
Test Scenario: Search Products by Keyword
User Story: As a Visitor, I want to search for products using keywords, so that I can quickly find specific items I'm looking for
Priority: Critical
Related Requirements: FR-003, FR-004

GIVEN the user is on the ecommerce website
  AND the search bar is visible in the header
  AND products exist in the catalog

WHEN the user enters a keyword in the search bar (e.g., "jeans")
  AND clicks the search button or presses Enter

THEN the system should display search results matching the keyword
  AND results should show products with "jeans" in:
    - Product name
    - Product description
    - Product keywords
  AND each result should display:
    - Product thumbnail
    - Product name
    - Price
    - Rating
  AND filter options should be available
  AND sorting options should be available
  AND the number of results found should be displayed

WHEN the user searches for a non-existent product (e.g., "xyz123abc")

THEN the system should display a "No results found" message
  AND suggest the user try different keywords
  AND optionally show featured/popular products

EXPECTED RESULT:
- Search successfully returns relevant products
- Search results are accurate and comprehensive
- Empty search results are handled gracefully
- Search performance is acceptable
- All product information is displayed correctly in results
