TEST SCRIPT: TS-097
Test Scenario: View Revenue Report by Period
User Story: As a Logged-in Admin, I want to generate revenue reports for specific time periods, so that I can analyze sales performance
Priority: High
Related Requirements: FR-021

GIVEN the admin is logged in to the admin panel
  AND there are completed orders in the system:
    - 2024-11-01: $94.48 (1 order)
    - 2024-11-05: $129.99 (1 order)
    - 2024-11-08: $215.97 (2 orders: $129.98 + $85.99)
    - 2024-11-10: $59.98 (1 order)
    - 2024-11-12: $184.95 (1 order)
  AND the admin navigates to "Statistics & Reports"

WHEN the admin clicks on "Revenue Report" option

THEN the system should display report filter options:
  - Report Period Type (dropdown):
    - Today
    - Current Week
    - Date Range (From - To)
    - Month (dropdown of months)
    - Year (dropdown of years)
  - "Generate Report" button
  - "Export" options (PDF, Excel)

WHEN the admin selects period type "Month"
  AND selects "November 2024"
  AND clicks "Generate Report"

THEN the system should:
  - Calculate total revenue for November 2024
  - Retrieve all completed orders from Nov 1-30, 2024
  - Sum order totals: $94.48 + $129.99 + $215.97 + $59.98 + $184.95 = $685.37
  - Display the revenue report showing:

    **Report Header:**
    - Report Title: "Revenue Report"
    - Period: "November 2024"
    - Generated Date: Current date
    - Generated By: Admin username

    **Summary Section:**
    - Total Revenue: $685.37
    - Total Orders: 6
    - Average Order Value: $114.23
    - Number of Customers: 5

    **Breakdown Section (Table):**
    | Date | Order Count | Daily Revenue |
    |------|-------------|---------------|
    | Nov 1 | 1 | $94.48 |
    | Nov 5 | 1 | $129.99 |
    | Nov 8 | 2 | $215.97 |
    | Nov 10 | 1 | $59.98 |
    | Nov 12 | 1 | $184.95 |

    **Visual Charts (Optional):**
    - Revenue trend line chart
    - Daily revenue bar chart

WHEN the admin selects period type "Date Range"
  AND sets From Date: "2024-11-08"
  AND sets To Date: "2024-11-12"
  AND clicks "Generate Report"

THEN the system should display revenue for that specific range
  - Total Revenue: $459.90 ($215.97 + $59.98 + $184.95)
  - Total Orders: 4
  - Average Order Value: $114.98

WHEN the admin selects period "Today"

THEN the system should display revenue for current date only
  - Show real-time data for today's orders
  - Update if new orders are placed

WHEN the admin clicks "Export as PDF"

THEN the system should:
  - Generate a PDF file containing the report
  - Include all report data and charts
  - Format professionally with company branding
  - Initiate file download: "Revenue_Report_November_2024.pdf"

WHEN the admin clicks "Export as Excel"

THEN the system should:
  - Generate an Excel spreadsheet
  - Include report data in structured format
  - Provide data suitable for further analysis
  - Initiate file download: "Revenue_Report_November_2024.xlsx"

EXPECTED RESULT:
- Revenue reports are accurately calculated for selected periods
- All completed orders are included in calculations
- Report data is presented clearly and professionally
- Multiple period types are supported (today, week, month, year, custom range)
- Visual charts enhance data understanding
- Export functionality works correctly for both PDF and Excel formats
- Reports can be used for business analysis and decision-making
